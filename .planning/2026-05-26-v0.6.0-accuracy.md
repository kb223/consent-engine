## Standard

If Kenneth's name is behind it, the standard is excellence. Specifically:

- **Zero false positives** in the Vendor Tracking Findings table. CMP cookies,
  session cookies, and CDN cookies must NOT appear there as "unknown legal risk".
- **CMP detection accuracy 100%** for the top 15 CMPs by market share.
- **CMP runtime introspection** for the 6 largest CMPs (OneTrust ✓,
  Cookiebot, CookieYes, Didomi, Usercentrics, Truyo).
- **Vendor library 5-10x larger** by ingesting from open-source DBs.
- **Consent event capture** covers all major CMPs' dataLayer broadcasts.
- **70+ tests stay green**. Ruff clean. Mypy strict-clean.
- **E2E verified** against ≥6 real sites covering different stacks.

## Self-annealing protocol

After each phase:
1. Run `uv run ruff check src/`
2. Run `uv run mypy src/`
3. Run `uv run pytest tests/ -x -q` (~4 min)
4. Run a smoke audit against a representative site
5. If any check fails: fix or revert, never commit broken state
6. Commit atomically with a descriptive message
